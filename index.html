Строки и столбцы

Строки и столбцы

Строки и столбцы - пространство между двумя горизонтальными или вертикальными линиями грида. Так же называют треком.

Строки и столбцы

                .grid {
                    grid-template-rows: track-size ...; //Размер строк 
                    grid-template-columns: track-size ...; //Размер столбцов
                }

track-size - проценты, фиксированный размер (например, пиксели) или fr

fr - часть свободного пространства в контейнере. Вычисляется после того, как рассчитаются все фиксированные элементы

Строки и столбцы

                .grid {
                    grid-template-rows: ;
                    grid-template-columns: ;
                }

Строки и столбцы. Repeat

                .grid {
                    grid-template-rows: 50px 50px 50px 50px 50px 50px 50px 50px 50px 50px;
                    grid-template-columns: 50px 1fr 1fr 1fr;
                }

Сокращаем! Повторяющиеся части можно заменить с помощью repeat()

                    .grid {
                        grid-template-rows: repeat(10, 50px);
                        grid-template-columns: 50px repeat(3, 1fr);
                    }

Строки и столбцы. Отступы

gap - формирует отступы между столбцами и строками

                gap: [row-gap] [column-gap]; //Горизонтальный отступ, потом вертикальный
                gap: [gap]; //По горизонтали и по вертикали одинаковый отступ
или 
                row-gap: [row-gap];
                column-gap: [column-gap];

Строки и столбцы. Отступы

                .grid {
                  grid-template-rows: 50px 1fr 1fr 50px;;
                  grid-template-columns: 20px 40px 20px;
                  gap: ;
                }

Строки и столбцы. Выравнивание

                .grid {
                    justify-items: start | end | center | stretch;
                    align-items: start | end | center | stretch;
                }

align-items выравнивает элементы по вертикальной оси.

justify-items выравнивает элементы по горизонтальной оси.

Линия

Линия

Разделяющие линии, которые формируют структуру грида. Бывают горизонтальными и вертикальными.

Линия

                .grid {
                  grid-template-columns: 200px 25% 1fr;
                }

Каждой линии присваивается отрицательное и положительное значение, начиная с 1 и -1.

Линия

Можно задать произвольное название (их может быть больше, чем одно).

                .grid {
                   grid-template-columns: [first one] 200px [second two два] 1fr [end];
                }

Ячейка

Ячейка

                .grid__item {
                    grid-column-start: [line-number] | [line-name];
                    grid-column-end: [line-number] | [line-name];
                    grid-row-start: [line-number] | [line-name];
                    grid-row-end: [line-number] | [line-name];
                }

Если не указано grid-column-start или grid-column-end, то размер ячейки будет равен одной колонке или строке.
grid-column-start:
grid-column-end:
grid-row-start:
grid-row-end:

Ячейка

grid-column - сокращение для grid-column-start + grid-column-end grid-row - grid-row-start + grid-row-end 

                .grid__item {
                  grid-column: [start-line] / [end-line];
                  grid-row: [start-line] / [end-line];
                }

Ячейка

grid-area - сокращение для grid-column-start + grid-column-end +
grid-row-start + grid-row-end 

                .grid__item {
                  grid-area: [row-start] / [column-start] / [row-end] / [column-end];
                }

Строки и столбцы. Выравнивание

Выравнивание элемента грида внутри ячейки: 

                    .grid {
                            align-self: start | end | center | stretch;
                            justify-self: start | end | center | stretch;
                        }

align-self выравнивает элементы по вертикальной оси.

justify-self выравнивает элементы по горизонтальной оси.

Ячейка. Выравнивание

place-self - сокращение для align-self + justify-self.

                place-self: [align-self] / [justify-self];
Если второе значение не будет задано, то первое будет использоваться для обоих значений.

Строки и столбцы. Автозаполнение

Если явно не задавать положение ячейки, то ячейки заполняются в соответствии с алгоритмом автозаполнения.

  • row - заполяет строки поочередно, добавляя новые при необходимости
  • column - заполняет колонки поочередно, добавляя новые при необходимости
  • dence - пытается заполнить дыры в сетке, если появляются мелкие элементы позже
 
                .grid {
                   grid-auto-flow: row | column | row dense | column dense;
                }

Область

Область. grid-template-areas

grid-template-areas определяет именнованные области в контейнере
Поместить элемент в область можно свойством grid-area. 

                grid-template-areas: "[grid-area-name] | . | none | ..."
  • . - пустая ячейка
  • none - нет никакой области
  • grid-area-namе - название области

Область. grid-template-areas

                        .grid {
                          display: grid;
                          grid-template-columns: 120px 90px 1fr 1fr;
                          grid-template-rows: repeat(4, 1fr);
                          grid-template-areas: "header header header header"
                                               "sidebar . main"
                                               "sidebar . main"
                                               "footer footer footer footer";
                        }
                    
                        .grid__header {
                          grid-area: header;
                        }
                    
                        .grid__main {
                          grid-area: main;
                        }
                    
                        .grid__sidebar {
                          grid-area: sidebar;
                        }
                    
                        .grid__footer {
                          grid-area: footer;
                        }
